Under Dr. Buzasi, I research using the program gPhoton. gPhoton is a Python program written to sift through GALEXs (Galaxy Evolution Explorer) time and location stamped photon data that was previously unviewable. Using the program, I go through star sources provided by Dr. Buzasi and develop light curves for the targets.
